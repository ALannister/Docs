### 1. 在windows下安装git
    安装Git-1.9.5-preview20150319.exe（安装路径不要有中文以及空格）

### 2. 安装可视化界面(可选)
    TortoiseGit-2.5.0.0-64bit.msi
    鼠标邮件，选择“TortoiseGit”->“Settings”，在“User Info”输入Name和Email
    
### 3. 生成公钥
    a. win+r --> Git Bash 
    b. $ ssh-keygen.exe
    c. 一路回车，将生成“c:/Users/XXX/.ssh/id_rsa.pub”
    d. 把id_rsa.pub重命名为id_rsa.XXX.pub, 例如id_rsa.jiangyikun.pub
    e. 将id_rsa.XXX.pub发送给梁安忠（飞秋或者邮件都可以）

### 4. 简单学习
    这是一个用于测试的repo你可以拿他来做做测试
    a. 克隆仓库
        git clone git@162.105.181.12:/home/git/project/sample.git
        或者使用tortoisegit，在某个文件夹，右键-->Git Clone...--> URL填写“git@162.105.181.12:/home/git/project/sample.git”--> 输入密码

    b. 修改文件并提交
        1. 在cpu目录下建立一个文件比如，姓名.txt
        2. 右键 Git Commit -> "Master"
        3. 填写Commit Message，并选择需要提交的文件，双击可以可视化的查看diff
    
    c. 点击push， ok
